FIDES Catalog API Gateway

This deployment proxies public catalog APIs to per-repository Vercel backends.

Discovery

  • GET /api/public/catalogs — which catalogs are configured

Credential catalog

  • GET /api/public/credentialtype
  • GET /api/public/credential-api-docs — OpenAPI (credential)
  • Swagger UI (credential)

Legacy URLs (redirect): /api/public/api-docs, /swagger.html

Organization catalog

  • GET /api/public/organization
  • GET /api/public/organization-api-docs — OpenAPI (organization)
  • Swagger UI (organization)

Issuer catalog

  • GET /api/public/issuer
  • GET /api/public/issuer-api-docs — OpenAPI (issuer)
  • Swagger UI (issuer)

Wallet catalog

  • GET /api/public/wallet — list (paginated)
  • GET /api/public/wallet/{orgId}/{walletId} — one wallet
  • GET /api/public/providers
  • GET /api/public/stats
  • GET /api/public/filter-options
  • GET /api/public/wallet-api-docs — OpenAPI (wallet)
  • Swagger UI (wallet)
